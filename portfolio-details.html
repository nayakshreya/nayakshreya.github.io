Find the Target Sum

Project information

  • Category: Mobile Application
  • Languages & Tools: JavaScript, React and React Native
  • Project date: February, 2021
  • Project URL: https://github.com/nayakshreya/game-app

Built a gaming app to find the target number by picking the correct numbers summing to the target number from a set of random numbers within 10 seconds otherwise, the participant loses the game and can play again. Generated game component with a random target number and random play numbers, the game is won if the sum of random play numbers is equal to the target number. The game is lost if the target sum was not found or the timer expired. Also, the user can play the game again with a different set of numbers.
